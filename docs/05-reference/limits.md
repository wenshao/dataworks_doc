# 限制与配额

## API 调用限制

### QPS 限制

| 版本 | 读 API QPS | 写 API QPS |
|------|-----------|-----------|
| 基础版 | 总计 ≤ 5 | 总计 ≤ 5 |
| 标准版 | - | - |
| 专业版 | - | - |
| 企业版 | 50 | 20 |

### 每日调用次数限制

| 版本 | 每日上限 |
|------|----------|
| 基础版 | 100 次 |
| 标准版 | 1,000 次 |
| 专业版 | 10,000 次 |
| 企业版 | 无限制 |

### 统计说明

- **统计范围**：每个主账号及其子账号在每个地域的累计值
- **计数规则**：调用成功和失败都计入限制次数
- **超限处理**：触发限制后 API 调用将失败

## 免费额度与计费

### 每月免费额度

| 版本 | 免费额度 | 超出后 |
|------|----------|--------|
| 基础版 | 3,100 次/月 | 不支持按量付费 |
| 标准版 | 31,000 次/月 | 不支持按量付费 |
| 专业版 | 310,000 次/月 | 不支持按量付费 |
| 企业版 | 1,000,000 次/月 | 0.3 元/万次 |

### 计费规则

- 只有**调用成功**才计费
- 免费额度每月 1 日重置
- 不同环境计费不同：
  - 公共云：0.3 元/万次
  - 政务云：0.6 元/万次
  - 金融云：0.57 元/万次

## API 读写分类

根据操作类型，API 分为读和写：

### 读 API 示例

- `GetFile` - 获取文件
- `ListFiles` - 文件列表
- `GetNode` - 获取节点
- `GetTable` - 获取表
- `ListTables` - 表列表

### 写 API 示例

- `CreateFile` - 创建文件
- `UpdateFile` - 修改文件
- `DeleteFile` - 删除文件
- `DeployFile` - 提交发布
- `RunNode` - 运行节点

## 调用建议

1. **合理控制并发**：避免触发 QPS 限制
2. **使用批量 API**：减少调用次数
3. **做好重试机制**：处理限流错误
4. **监控使用量**：通过控制台查看额度使用情况

## 查看使用情况

登录 DataWorks 控制台 → 开放平台 → 开发者后台

可以查看：
- 免费额度使用情况
- API 调用次数统计
- API 调用明细
