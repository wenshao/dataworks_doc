# DataWorks 功能特性

## 核心功能模块

### 1. 数据集成

DataWorks 数据集成支持离线和实时数据同步，覆盖 50+ 数据源类型。

**支持的数据源类型：**
- 关系型数据库：MySQL、Oracle、SQL Server、PostgreSQL 等
- 大数据引擎：MaxCompute、Hive、Kudu 等
- NoSQL 数据库：MongoDB、Redis、HBase 等
- 消息队列：Kafka、RocketMQ 等
- 文件存储：OSS、HDFS、FTP 等
- 其他：Elasticsearch、Hologres、AnalyticDB 等

**核心能力：**
- 离线同步：支持批量数据传输
- 实时同步：支持实时数据流处理
- 整库同步：支持整库迁移和同步
- Serverless 同步：无需管理计算资源

### 2. 数据开发

提供可视化的数据开发环境，支持多种开发语言。

**支持的语言：**
- SQL（ODPS SQL、MySQL SQL、Oracle SQL 等）
- Shell
- Python
- Java UDF
- Spark
- MapReduce

**功能特性：**
- 可视化节点开发
- 工作流编排
- 调度配置
- 代码版本管理
- 在线调试

### 3. 数据建模

智能数据建模提供 DDM（Dimensional Data Modeling）建模能力。

**核心功能：**
- 数仓规划：分层设计（ODS/DWD/DWS/ADS）
- 数据标准：数据命名规范、字段标准
- 维度建模：维度表设计、事实表设计
- 数据指标：原子指标、派生指标管理

### 4. 运维中心

提供全方位的任务运维能力。

**功能特性：**
- 任务监控：实时监控任务运行状态
- 补数据：支持历史数据补跑
- 智能报警：支持多种报警方式（钉钉、短信、邮件）
- 实例管理：查看和管理任务实例
- 基线管理：保障核心任务按时产出

### 5. 数据治理

提供全面的数据治理能力。

**功能模块：**
- **数据地图**：数据资产全景图
- **数据质量**：质量规则配置、监控和告警
- **数据资产治理**：资产价值评估、治理健康度评分
- **安全中心**：权限管理、敏感数据保护
- **数据保护伞**：数据脱敏、访问审计
- **审批中心**：数据开发审批流程

### 6. 数据服务

将数据快速生成 API，实现数据服务化。

**功能特性：**
- 向导式 API 生成
- 支持 RESTful 风格
- API 权限控制
- API 调用统计
- 数据推送服务

### 7. 数据分析

提供即席查询和深度分析能力。

**功能特性：**
- SQL 查询：支持多引擎 SQL 查询
- Notebook：交互式分析环境
- 数据目录：统一的数据元数据管理
- 数据探查：数据分布分析
- 可视化：图表展示

### 8. 开放平台

提供开放的集成能力。

**开放能力：**
- **OpenAPI**：100+ API，支持所有核心功能
- **OpenEvent**：事件订阅机制，实时获取系统事件
- **Extensions**：扩展程序机制，支持自定义逻辑

## 产品版本

| 版本 | 适用场景 | 核心能力 |
|------|----------|----------|
| 基础版 | 小团队入门 | 基本数据开发与调度 |
| 标准版 | 中小企业 | 完整数据开发能力 |
| 专业版 | 中大型企业 | 数据治理与智能运维 |
| 企业版 | 大型企业 | 全功能 + 企业级支持 |

## 新版 Data Studio

DataWorks 新版 Data Studio 提供更好的开发体验：

- **目录管理**：更直观的文件组织方式
- **个人开发环境**：独立的开发计算资源
- **数据目录**：统一的数据资产浏览
- **工作流增强**：更灵活的工作流编排
