# 什么是 DataWorks

## 产品简介

大数据开发治理平台 DataWorks 基于MaxCompute/EMR/MC-Hologres等大数据计算引擎，为客户提供专业高效、安全可靠的一站式大数据开发与治理平台，自带阿里巴巴数据中台与数据治理最佳实践，赋能各行业数字化转型。

## 核心价值

- **一站式开发治理平台**：提供从数据集成、开发、调度、运维到治理的全链路能力
- **阿里巴巴最佳实践**：内置集团内部多年积累的数据中台与数据治理经验
- **大规模生产验证**：每天阿里巴巴集团内部有数万名数据/算法工程师使用，承担集团 99% 数据业务构建

## 支持的计算引擎

- MaxCompute（原 ODPS）
- E-MapReduce（EMR）
- Hologres
- 其他主流大数据计算引擎

## 产品定位

DataWorks 定位为企业级大数据开发治理平台，主要面向：

- 数据开发工程师
- 数据分析师
- 数据治理专家
- 算法工程师

## 核心功能模块

| 模块 | 功能描述 |
|------|----------|
| 数据集成 | 支持离线/实时数据同步，覆盖 50+ 数据源 |
| 数据开发 | 可视化节点开发、工作流编排、调度配置 |
| 数据治理 | 数据地图、数据质量、数据资产治理、安全中心 |
| 运维中心 | 任务监控、补数据、智能报警 |
| 数据服务 | 快速生成 API，实现数据服务化 |
| 数据分析 | SQL 查询、Notebook 分析、数据可视化 |
| 开放平台 | OpenAPI、OpenEvent、扩展程序 |
