# 阿里云 DataWorks 文档

> 大数据开发治理平台 DataWorks 官方文档整理

## 📖 简介

阿里云 DataWorks 是基于 MaxCompute/EMR/MC-Hologres 等大数据计算引擎的一站式大数据开发与治理平台，自带阿里巴巴数据中台与数据治理最佳实践。每天阿里巴巴集团内部有数万名数据/算法工程师正在使用 DataWorks，承担集团 99% 数据业务构建。
